About Stak Suite

This website was made by Misha Pavlov, with some inspiration taken from an original idea by James Churchill.
This project is currently my work as part of an NCEA assignment. Following NCEA submission in September 2022 there is substantial scope for further collaborative project development, such as implementation of reverse engineering features, virtual image features and an online SSH environment.
